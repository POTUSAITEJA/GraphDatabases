## Overview

Apache Marmotta was an open-source platform for Linked Data and the W3C Linked Data Platform recommendation. It supported full read-write Linked Data, SPARQL 1.1, LDP, lightweight reasoning, and basic security in a single Java web application. It was one of the reference projects implementing the LDP recommendation.

The project was retired to the Apache Attic in November 2020, which means it is no longer actively maintained. It is included in the catalog as a historical reference for teams that encounter Marmotta deployments or need context on the evolution of W3C LDP implementations.

## Vector strategy — None

No vector capability. Marmotta predates the modern vector-database era and the project ended before vector search became a mainstream concern for triple stores.
