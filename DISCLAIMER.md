# Disclaimer

## Purpose

This repository is an **independent, community-maintained reference guide** for RDF triple stores and their vector search capabilities. It is intended to help engineers, architects, and researchers make informed decisions when evaluating graph database technologies.

- Inclusion or exclusion of any product does **not** constitute an endorsement or criticism.
- All assessments are based on **publicly available information** at the time of writing and are presented in good faith.
- Different tools are designed for different use cases. A feature marked as "not currently available" simply means it was not found in official documentation at the time of review — it does not imply the tool is unsuitable for its intended purpose.

### Accuracy & Currency

- The database landscape evolves rapidly. Information in this repository may become outdated.
- We make every effort to verify claims against **primary sources** (official documentation, GitHub repositories, published papers). However, errors may occur.

### Maturity Assessments

- Maturity ratings reflect the **observable evidence** of production deployments, release history, community adoption, and operational documentation — not a judgment of code quality or engineering capability.

- Projects labeled "preview" or "experimental" are identified based on their **own official communications** (e.g., README labels, version numbers, documentation status).
